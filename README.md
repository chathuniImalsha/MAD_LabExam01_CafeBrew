# ☕ Café Brew

A modern and elegant **Coffee Shop Mobile App UI/UX Design** created using **Figma**. Café Brew delivers a seamless coffee ordering experience through an intuitive interface, interactive prototype, and visually appealing design inspired by premium coffee aesthetics.

---

## 📖 About the Project

Café Brew is a mobile application UI/UX design project developed as part of the **Mobile Application Development** module at **SLIIT**.

The application is designed to provide users with a smooth and enjoyable coffee ordering experience, from onboarding to checkout. The design focuses on simplicity, usability, modern visual aesthetics, and an intuitive user experience.

---
